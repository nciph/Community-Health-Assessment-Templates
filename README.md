# Community Health Assessments Templates (CHA)
This repository contains templates and resources for conducting Community Health Assessments (CHA). 

## What's in this repository?
If your Community Health Assessment (CHA) is done in Qualtrics, you can use the following resources: 
### Access Qualtrics Data and Detect Fraudulent Responses
- **Accessing Qualtrics Data**: use-qualtrics-api.R - A script for retrieving data from Qualtrics surveys.
- **Qualtrics Fraud Detection Code**: qualrics-fraud-detection.R - A script for detecting fraudulent responses in Qualtrics surveys.
### Apply Survey Weights to CHA Data
- **Survey Weighting Tutorial**: The survey weighting tutorial is built in quarto and provides step-by-step instructions on how to apply survey weights to your CHA data.The published version of the tutorial can be accessed [here](https://nciph.github.io/Community-Health-Assessment-Templates/).
The tutorial was developed using quarto, and you can find the source code for the tutorial in the repository.
- _quarto.yml: this file contains the configuration for the quarto project, including settings for rendering and publishing the tutorial.
- index.qmd: this file contains the content of the tutorial, including text, code chunks, and visualizations. You can edit this file to customize the tutorial for your specific needs.

You can find survey weighting R scripts used in the tutorial in the `survey-weighting` folder.
